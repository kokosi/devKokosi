문제해결

좀 더 많은 문제들에 대한 해결 방법은 FAQ 또는 고객센터를 통해 안내 받으시기 바랍니다

FAQ 바로가기

질문1. 브라우저에서 쿠키/캐시 삭제는 어떻게 해야 하나요?

  • 브라우저의 쿠키/캐시는 왜 삭제해야 하나요?

    사용하고 계신 브라우저(IE, FF, GC 등)가 임의적으로 보관하고 있는 게임 이용 데이터가 간혹 게임 서버 내의 데이터와 올바르게
    연결되지 않을 경우 오류가 발생할 수 있습니다.(점검 직후 등) 이 경우 아래의 과정을 통해 브라우저의 쿠키/캐시 삭제를 진행해
    주시면 정상적으로 게임을 이용하실 수 있습니다.

  • 인터넷 익스플로러(Internet Explorer)

    1. 1. 익스플로러 메뉴에서 [도구-인터넷 옵션]으로 이동합니다.
    2. 2. [일반] 탭의 [검색 기록]에 있는 [삭제] 버튼을 클릭합니다.
    3. 3. 모든 항목에 체크 후 [삭제] 버튼을 클릭 합니다.

  • 구글 크롬(Google Chrome)

    1. 1. 주소창 우측의 [옵션-방문기록]을 삭제 합니다.
    2. 2. [인터넷 사용기록 모두 삭제]를 클릭합니다.
    3. 3. 모든 항목에 체크 한 뒤 [인터넷 사용정보 삭제]를 클릭 합니다.

  • 스윙 브라우저(Swing-Browser)

    1. 1. 주소창 우측의 [스윙 브라우저 설정 및 관리]를 클릭합니다.
    2. 2. 방문 기록 선택 후 [인터넷 사용 기록 모두 삭제]를 클릭합니다.
    3. 3. 다음 항목 삭제를 [전체]로 선택 후 모든 항목에 체크합니다.
    4. 4. [인터넷 사용정보 삭제]를 클릭합니다.

  • 모질라 파이어폭스(Mozilla Firefox)

    1. 1. 파이어폭스 메뉴에서 [도구-사용기록 삭제]를 클립합니다.
    2. 2. 모든 항목에 체크 후 [지금 삭제]를 클릭 합니다.

질문2. IE 11에서 사이트 이용 시 오류가 발생합니다.

  • Internet Explorer 11버전의 경우 기존 10버전 이하와 달리 호환성에서 여러 문제가 발생하는 것으로 파악이 됩니다.
    이 경우 아래의 가이드를 참고하여 해결이 가능합니다.
    1. 1. IE 의 도움말을 실행한 후 Internet Explorer 정보를 누릅니다.
    2. 2. 현재 오류가 발생하고 있는 IE의 버전이 11이 맞는지 확인합니다.
    3. 3. IE의 버전이 11이 맞을 경우 IE의 메뉴에서 도구 > 호환성 보기 설정으로 이동합니다.
    4. 4. 호환성 보기 설정 화면에서 ssengame.com을 추가합니다.
    5. 5. 실행중인 모든 브라우저를 닫은 후 다시 사이트에 접속합니다.
